A short overview of the local web dashboard running a complete drone-video analysis flow. The cases below show the same interface across multiple drone scenes.

Abstract

Long drone videos contain both spatial information and temporal events, but reviewing them manually is slow. This project provides a local web dashboard that turns a user-uploaded video into three result views: a LingBot-MAP reconstruction for understanding scene structure, a TriMamba summary for quickly reviewing the trimmed video, and Lighthouse retrieval results for finding query-relevant moments.

The page below is organized by input video. Each case shows the original/trimmed footage, the reconstruction result, the summarization result, and the query-based retrieval result.
